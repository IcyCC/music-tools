# 🎹 音乐学习工具

一套用于学习和弦听辨的交互式网页工具。

## 📚 包含工具

### 1. 和弦听辨入门课程
**路径**: `/chord-course/`

从零开始学习和弦听辨的互动课程：
- **第1课**: C、F、G 三个大调和弦
- **第2课**: Am、Dm、Em 三个小调和弦
- 包含：概念讲解 + 音频演示 + 互动练习

### 2. 和弦听力训练器
**路径**: `/chord-trainer/`

专业的和弦听力训练工具：
- 4种练习模式（识别走向、单个和弦、调性识别、听写模式）
- 5级难度系统
- 50+种常见和弦走向
- 5种乐器音色
- 进度追踪和成就系统
