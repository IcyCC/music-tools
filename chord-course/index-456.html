🎷 爵士乐的核心：ii-V-I 走向

学会这个走向，你就拿到了爵士乐的入场券！

🎷 为什么 ii-V-I 这么重要？

如果把流行音乐比作"白米饭"，那爵士乐就是"法式大餐"——更复杂、更精致。

而 ii-V-I 就是爵士乐的"盐"——几乎每首爵士歌都有它！

据统计，85%以上的爵士标准曲都使用 ii-V-I 或其变体。

📚 什么是 ii-V-I？

在C大调中：

  • ii = Dm（二级，小调和弦）
  • V = G（五级，大调/属和弦）
  • I = C（一级，主和弦）

所以 ii-V-I 在C大调就是：Dm → G → C

Dm → G → C

ii - V - I (Dm - G - C)

🎵 为什么 ii-V-I 这么好听？

第一步：Dm (ii) - "预备"

Dm是小调和弦，它不是紧张，而是"准备"。 就像起跑前的"预备——"

第二步：G (V) - "紧张"

从Dm到G，紧张感急剧上升！ 就像"预备——跑！"的那个"跑！"

第三步：C (I) - "解决"

最后回到C，完美的解决！ 那种"啊~终于到了"的感觉。

🔄 ii-V-I vs I-IV-V：有什么区别？

流行/民谣风格
简单直接

VS

爵士风格
更流畅优雅

关键区别：ii-V-I从小调开始，所以更"柔和"，而I-IV-V从大调开始，更"直接"。

ii-V-I像"滑行"，I-IV-V像"跳跃"。

🔧 ii-V-I 的扩展：vi-ii-V-I

扩展

Am → Dm → G → C

在ii-V-I前面加一个vi(Am)，就变成了4小节的完整走向！

  1. Am - 更长的准备
  2. Dm - 继续准备
  3. G - 紧张！
  4. C - 解决！

📝 第4课总结

💡 ii-V-I 听辨技巧

  1. 听第一个和弦：如果是小调(Dm)，可能是ii-V-I
  2. 听流畅感：ii-V-I很"滑"，像下楼梯
  3. 听解决感：最后回到C时有强烈的"完成感"
第5课

🎸 蓝调基础：12小节蓝调

这是摇滚、布鲁斯、爵士的共同祖先！

🎸 什么是蓝调？

蓝调(Blues)起源于美国南方黑人音乐，是摇滚、爵士、R&B的"祖先"。

蓝调的核心是一个12小节的结构，叫做"12小节蓝调"。

学会了这个，你就能理解90%的早期摇滚和布鲁斯！

📊 12小节蓝调的结构

前4小节
C C C C
在主和弦上"陈述主题"
中间4小节
F F C C
到F"展开"，再回到C
最后4小节
G F C G
高潮+回归+收尾

简化记忆：

  • 第1句 (1-4小节): C C C C — "我想你了"
  • 第2句 (5-8小节): F F C C — "真的想你了"
  • 第3句 (9-12小节): G F C G — "哇我好想你啊！"

🎵 逐段感受

第一段：C-C-C-C "我想你了"

4个小节都是C，非常稳定。 这是蓝调的"开场白"，歌手在这里唱出"主题"。

第二段：F-F-C-C "真的想你了"

到F，情绪升级！ 歌手重复第一段的旋律，但和弦变了，所以感觉更强烈。

第三段：G-F-C-G "哇我好想你啊！"

这是高潮！G是紧张的，然后F-C解决，最后G留下悬念... 这最后两个和弦(G)叫做"turnaround"，为下一轮做准备。

Johnny B. Goode Chuck Berry

🎤 更多流行歌曲走向

学完这些，你就是和弦走向大师了！

🎯 这一课你将学到

  • Axis进行 - 现代流行乐的"万能走向"

  • Andalusian进行 - 摇滚/弗拉明戈风格

🎸 走向1：Axis 进行 (vi - IV - I - V)

🔥 现代流行

Am → F → C → G

📖 特点

从小调(Am)开始，但最后结束在大调(G)——"悲伤开始，希望结尾"。

和"悲伤进行"(Am-F-C-G)几乎一样，但情感更积极，因为最后一个和弦G暗示"会解决到C"。

📺 经典歌曲

With or Without You U2
Be the One Dua Lipa

💫 走向2：敏感进行 (I - iii - vi - IV)

💫 情绪细腻

C → Em → Am → F

📖 特点

用iii(Em)代替Canon中的V(G)，情感更内敛、敏感。

Canon: C-G-Am-F (稳定→紧张→忧伤→扩展)

敏感: C-Em-Am-F (稳定→流动→忧伤→扩展)

区别在于第2个和弦：G更紧张，Em更流动。

🔄 对比 Canon 和 敏感进行

Canon更"有动力"，敏感更"内敛"

📺 经典歌曲

No Woman No Cry Bob Marley

🔥 走向3：Andalusian 进行 (i - bVII - bVI - V)

🔥 摇滚/弗拉明戈

Am → G → F → E (或 E7)

📖 特点

这是西班牙弗拉明戈的经典走向，也是很多摇滚/金属歌曲的基础。

特点：全程下行，从A到G到F到E，像"下楼梯"一样。

最后一个和弦E(E7)非常紧张！会强烈想要回到Am。

📺 经典歌曲

Hit the Road Jack Ray Charles
Sultans of Swing Dire Straits
Stairway to Heaven Led Zeppelin (部分)
