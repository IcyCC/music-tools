🎹 和弦听辨入门
第1课

认识三个最重要的和弦：C、F、G

这三个和弦是流行音乐的基础，学会它们你就能听懂大部分歌曲！

📚 什么是和弦？

和弦就是三个或更多音符同时发声的组合。

想象一下：如果旋律是"一句话"，那和弦就是这句话的"背景情绪"。

E C G
= C 和弦

🎵 认识 C、F、G 三个和弦

C 和弦 - "家"的感觉

主和弦 (Tonic)
C + E + G

C和弦是C大调的"主和弦"，也是这个调的"家"。 听到它，你会感觉"完成了"、"稳定了"。

🎵 仔细听：明亮、稳定、像回到家的感觉

📺 真实歌曲例子

"小星星" 开头第一个和弦就是C。

"Let It Be" (Beatles) 主歌开始也是C和弦。

F 和弦 - "向外走"的感觉

下属和弦 (Subdominant)
F + A + C

F和弦叫做"下属和弦"，它让你感觉"离开了家，在旅途中"。 它既不是终点，也不是最紧张的地方，而是"中间状态"。

🎵 仔细听：像是踏上旅途，有期待感

G 和弦 - "想要回家"的感觉

属和弦 (Dominant)
G + B + D

G和弦叫做"属和弦"，它是最紧张的和弦！ 听到它，你会强烈地"想要回到C"——这种感觉叫做"张力"。

🎵 仔细听：紧张！想要"解决"到C！

💡 什么是"解决" (Resolution)？

解决就是"紧张→放松"的过程。

  • C (家) → 稳定
  • F (旅途) → 开始变化
  • G (危机点) → 最紧张！
  • C (回家) → 解决！满足！

🎵 亲自感受"解决"

🎯 练习：听辨 C、F、G

第2课

三个小调和弦：Am、Em、Dm

学会这些，你就能感受"忧伤"和"深沉"的音乐情绪了！

🎭 大调 vs 小调

明亮、开心 ☀️

VS

忧伤、深沉 🌙

🎵 认识 Am、Dm、Em

Am 和弦 - "忧伤的家"

小调主和弦
A + C + E

Am是C大调的关系小调，悲伤但稳定。

Dm 和弦 - "更悲伤"

小调下属和弦
D + F + A

Dm比Am更加悲伤，像是在哭泣。

Em 和弦 - "流动的忧伤"

小调属和弦
E + G + B

Em忧郁但不沉重，有向前流动的感觉。

🎯 练习：听辨 Am、Dm、Em

第3课

🎸 流行歌曲常见和弦走向

学会这些经典走向，你就能听懂90%的流行歌！

📚 什么是和弦走向？

和弦走向（Chord Progression）就是一连串按顺序播放的和弦。

就像讲故事有"起承转合"，和弦走向也有它的"剧情"！

同一首歌，换一个走向，情绪就完全不同。

C → G → Am → F

这就是传说中的 "Canon 进行"！

🏆 走向1：Canon 进行 (I - V - vi - IV)

🔥 最流行

C → G → Am → F

📖 为什么这么好听？

这个走向有一个"循环感"——永远不会真正"结束"，总是在流动。

  1. C (I) - 开始，稳定
  2. G (V) - 制造紧张
  3. Am (vi) - 转折，忧伤感
  4. F (IV) - 扩展，然后回到C...

它像一个"情感过山车"：稳定→紧张→忧伤→期待→稳定...

📺 用这个走向的经典歌曲

Let It Be Beatles
Don't Stop Believing Journey
We Are Young fun.
逆光 孙燕姿
十年 陈奕迅

💡 试着弹奏或听这些歌，感受同一个走向！

📻 走向2：50年代进行 (I - vi - IV - V)

📺 经典

C → Am → F → G

📖 和Canon有什么不同？

注意！这个走向和Canon和弦一样，但顺序不同！

Canon：C→G→Am→F（稳定→紧张→忧伤→扩展）

50年代：C→Am→F→G（稳定→忧伤→扩展→紧张）

最后停在G，有更强的"未完成感"，想回到C！

📺 用这个走向的经典歌曲

Stand By Me Ben E. King
Every Breath You Take The Police
Unchained Melody Righteous Brothers

🔄 对比两个走向

仔细感受：Canon更"流动循环"，50年代更"期待结束"

😢 走向3：悲伤进行 (vi - IV - I - V)

😢 悲伤

Am → F → C → G

📖 为什么听起来很悲伤？

这个走向从小调和弦(Am)开始，所以一开始就是忧伤的情绪！

  1. Am - 忧伤的开始
  2. F - 扩展忧伤
  3. C - 短暂的明亮
  4. G - 又回到紧张

整个走向像是"在悲伤中挣扎"。

📺 用这个走向的经典歌曲

Someone Like You Adele
Apologize OneRepublic
童话 光良

🎸 走向4：最简单的三和弦 (I - IV - V)

⭐ 入门

C → F → G

📖 为什么先学这个？

只有三个和弦：主(C)、下属(F)、属(G)，是所有走向的基础！

很多民谣、布鲁斯、摇滚都用这个。

📺 用这个走向的歌曲

La Bamba Ritchie Valens
Twist and Shout Beatles
朋友 周华健

📝 第3课总结

🔄
Canon进行 (C-G-Am-F)

最流行！循环感，永不结束的感觉。

📺
50年代进行 (C-Am-F-G)

经典！期待感，想要回到C。

😢
悲伤进行 (Am-F-C-G)

忧伤！从小调开始，在悲伤中挣扎。

⭐
三和弦 (C-F-G)

最简单！所有走向的基础。

💡 听辨技巧

  1. 听第一个和弦：从C开始=明亮，从Am开始=悲伤
  2. 听"回家的感觉"：Canon循环，50年代期待
  3. 哼唱旋律：很多歌的旋律会告诉你走向
  4. 多听多练：熟悉了就能"秒认"
