🎹 和弦听辨入门
第1课

认识三个最重要的和弦：C、F、G

这三个和弦是流行音乐的基础，学会它们你就能听懂大部分歌曲！

📚 什么是和弦？

和弦就是三个或更多音符同时发声的组合。

想象一下：如果旋律是"一句话"，那和弦就是这句话的"背景情绪"。

E C G
= C 和弦

🎵 认识 C、F、G 三个和弦

C 和弦 - "家"的感觉

主和弦 (Tonic)
C + E + G

C和弦是C大调的"主和弦"，也是这个调的"家"。 听到它，你会感觉"完成了"、"稳定了"。

🎵 仔细听：明亮、稳定、像回到家的感觉

📺 真实歌曲例子

"小星星" 开头第一个和弦就是C，"一闪一闪亮晶晶"——那个明亮稳定的声音。

"Let It Be" (Beatles) 主歌开始也是C和弦。

F 和弦 - "向外走"的感觉

下属和弦 (Subdominant)
F + A + C

F和弦叫做"下属和弦"，它让你感觉"离开了家，在旅途中"。 它既不是终点，也不是最紧张的地方，而是"中间状态"。

🎵 仔细听：像是踏上旅途，有期待感，不那么稳定

📺 真实歌曲例子

"小星星" 第二句"满天都是小星星"开头是F，感觉像在扩展。

"Country Roads" "West Virginia" 那个向上走的感觉。

G 和弦 - "想要回家"的感觉

属和弦 (Dominant)
G + B + D

G和弦叫做"属和弦"，它是最紧张的和弦！ 听到它，你会强烈地"想要回到C"——这种感觉叫做"张力"。

🎵 仔细听：紧张！不满足！感觉要"解决"到某个地方！

📺 真实歌曲例子

"小星星" "挂在天空放光明"最后那个和弦，感觉没结束？那就是G！

"Twist and Shout" 最后那句之前，紧张感达到顶点。

💡 什么是"解决" (Resolution)？

解决就是"紧张→放松"的过程。

想象一部电影：

  • C (家) → 一切安好，剧情稳定
  • F (旅途中) → 开始冒险，事情变化
  • G (危机点) → 高潮！最紧张！你屏住呼吸
  • C (回家) → 啊~解决了！满足！

音乐就是在不断创造"紧张"然后"解决"，让你感觉像坐过山车一样！

🎵 亲自感受"解决"

第一个按钮：G紧张 → C解决，是不是感觉"啊~舒服了"？
第二个按钮：只停在G，是不是感觉"别停啊！要解决啊！"？

📝 第1课总结

🏠
C 和弦 (主和弦)

稳定、完成、家。明亮的大调声音。

🚶

第2课

三个小调和弦：Am、Em、Dm

学会这些，你就能感受"忧伤"和"深沉"的音乐情绪了！

🎭 大调 vs 小调：情绪的差异

明亮、开心、阳光 ☀️

VS

忧伤、深沉、内省 🌙

小调和弦比大调和弦少一个半音。

C大调：C-E-G（大三度）

Am小调：A-C-E（小三度）

就是这一个半音的差别，决定了"快乐"还是"忧伤"！

🎵 认识 Am、Em、Dm 三个小调和弦

Am 和弦 - "忧伤的家"

小调主和弦 (Minor Tonic)
A + C + E

Am和弦是A小调的"主和弦"。 它和C大调是"亲戚"（关系小调），但情绪完全不同！ C是阳光，Am是月光。

🎵 仔细听：悲伤但不绝望，内省、温柔

📺 真实歌曲例子

"Someone Like You" (Adele) 开头那个和弦，一听到就想哭？那就是Am的感觉。

"Hurt" (Johnny Cash) 整首歌都是Am的情绪。

Dm 和弦 - "悲伤的旅途"

小调下属和弦 (Minor Subdominant)
D + F + A

Dm和弦是小调世界的"旅途"和弦。 它比Am更加悲伤，感觉像是在哭泣。 很多悲伤歌曲会用Dm→Am的进行。

🎵 仔细听：比Am更悲伤、更沉重

📺 真实歌曲例子

"Sultans of Swing" (Dire Straits) 那个经典吉他riff开头。

"小星星"的小调版本会从Am开始，第二句变成Dm。

Em 和弦 - "小调的紧张"

小调属和弦 (Minor Dominant)
E + G + B

Em和弦有趣的地方是——它本身是一个小调和弦， 但在C大调里，它经常被用作"连接者"，不像G那么紧张，但有一种流动感。

🎵 仔细听：忧郁但不沉重，有一种向前流动的感觉

📺 真实歌曲例子

"Hotel California" (Eagles) 那个经典进行 Am-Em... 里的Em。

"Let It Be" 进行 C-G-Am-F 也可以改成 C-Em-Am-F。

📊 大调和弦 vs 小调和弦 速查表

功能
大调 (快乐)
小调 (忧伤)
主和弦 (家)
明亮稳定
悲伤稳定
下属和弦 (旅途)
向外扩展
更加悲伤
属和弦 (紧张)
强烈紧张
流动感

💡 小调的"解决"感觉不同

大调的解决是"完成！开心！"

小调的解决是"接受。平静。"

就像：

  • 大调解决 = "问题解决了！🎉"
  • 小调解决 = "我接受这个悲伤了。🌧️"

🎵 感受小调的"解决"

Dm→Am：悲伤的接受，像雨过天晴后的宁静。
G→C：快乐的完成，像完成了一项任务！

📝 第2课总结

🌙
Am 和弦 (小调主和弦)

悲伤但稳定，像月光下的思考。

🌧️
Dm 和弦 (小调下属和弦)

比Am更悲伤，像哭泣的旅程。

🌊
Em 和弦 (小调属和弦)

流动的忧伤，向前推进。

💡 听辨技巧

  1. 先判断大调还是小调：明亮 vs 忧伤
  2. 再判断功能：稳定(家) vs 旅途 vs 紧张
  3. 相信第一感觉：不要想太多！
